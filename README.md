# Stremio Playback Shortcut

A Stremio plugin that adds keyboard shortcuts for controlling video playback speed.

## Shortcuts

- **Shift + <** - Decrease playback speed(down to 0.25x)
- **Shift + >** - Increase playback speed(up to 4x)
